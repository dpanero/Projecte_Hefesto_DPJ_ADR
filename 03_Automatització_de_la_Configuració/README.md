# Objectiu del projecte

Implementar un sistema de gestió de configuració que permeti:

- automatitzar configuracions
- reduir errors manuals
- mantenir sistemes en un estat conegut
- facilitar el desplegament d'infraestructura

Aquest tipus d'eines permet gestionar múltiples sistemes des d'una única plataforma.
